Angular DateParser Demo

This parser converts date and time strings into JavaScript Date objects.

Basic

Date and time

Selected date: {{date | date:'MMM d, y h:m'}}

Code
<input type="text" ng-model="date" date-parser="dd.MM.yyyy HH:mm" />

Using Angular Date Formats

Use all the shortname formats provided by Angular.

medium

short

fullDate

longDate

mediumDate

shortDate

mediumTime

shortTime

Code
